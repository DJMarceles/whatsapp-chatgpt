# @kollie/mcp-server-core

Shared utilities and default tools for building MCP-compliant servers in the Kollie ecosystem.

## Contents
- `src/index.ts` – lightweight registry helpers for registering tool definitions.
- `tools/health.ping.ts` – canonical health check tool.

## Scripts
- `pnpm build` – compile TypeScript to ESM with declarations.
- `pnpm test` – placeholder suite.
